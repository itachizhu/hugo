使用Hugo搭建个人博客

 Posted on November 15, 2018  |  itachizhu

网络上已经有很多这样的教程了，为什么我还要自己写一个呢？原因有二：

  1. 犯贱，因为由于版本和系统的不同，我在追新的同时就免不了踩很多的坑，而再踩坑之后会让我很不爽，于是我就跟它硬刚，刚完之后遍体鳞伤之余我便记录了下来。
  2. 虽然不知道什么时候开始，莫名地喜欢写代码了，也喜欢自己一个静静地想一些技术性的问题，但其实我的骨子里就是一个网络写手，我还曾经不止一次地梦想过有那么一天，我写了一部网络小说，然后，火了。所以在这里慢慢完善的同时我是想记录一些自己喜欢的诗人，作家的文章。

所以，我搭建了这么一个类似博客那样的东西，与技术无关不接受吐槽放着那些我的随笔和我喜欢的文章。当我有这样的想法的时候，我本身又不太喜欢网络上那些繁杂而不自由的东西，于是我在找寻和丰富自己技术之余有了这个东东。

创建账户，设置页面服务

要么说为什么我是那样崇拜linux的创始人Linus Torvalds呢？他一直是我心目中最高级别的神并且没有之一。出自他手的所有东西都是神器。而今天我们要使用的就是他创造的神器之一Github Pages。

首先我们在我们的github账户中创建一个repository，然后命名成<<您的账户名>>.github.io（如果系统提示不让这样创建就先随便命名一个，后面可以改的）。

然后到资源库链接中点击Settings，进入设置界面，搜索”Github Pages”到相关设置的地方，选择对应的分支版本，点击Save即可。此时不要点击对应的链接，因为没有内容肯定是404页面的。

最后需要在对应的repository下面的根路径中创建index.html文件，输入:

<!DOCTYPE html>
<html lang="en" itemscope itemtype="http://schema.org/WebPage">
  <head>
  <meta charset="utf-8" />
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0">
  <title>Golden Mountain - Golden Mountain</title>
  </head>
  <body>Hello World!</body>
</html>

此时你再访问之前设置的地址如：https://itachizhu.github.io/的话，应该对应的demo页面就出来了：

使用hugo

安装hugo

安装hugo很简单，到下载页面去下载对应的系统可执行包即可hugo的github页面，当然如果你使用的是mac系统，而且安装了brew的话，那么恭喜你，你可以使用一条简单的命令即可完成安装：

$ brew install hugo
使用hugo创建网站

首先建立自己的站点，这里命名为quickstart

$ hugo new site quickstart

然后进入该路径

$ cd quickstart

该目录下你可以看到以下几个目录和config.toml文件

▸ archetypes
▸ content
▸ data
▸ layouts
▸ resources
▸ static
▸ themes
config.toml

config.toml是网站的配置文件，包括baseurl，title，copyright等等网站参数。

这几个文件夹的作用分别是：

  • archetypes：包括内容类型，在创建新内容时自动生成内容的配置
  • content：包括网站内容，全部使用markdown格式
  • data：存放数据和一些扩展配置，比如json文件
  • layouts：包括了网站默认模板，决定了网站默认的内容呈现方式
  • resources：资源目录
  • static：包括了css，js，fonts，media等，决定网站默认的外观
  • themes：存放主题皮肤，网上有很多牛人和爱好者提供了各种好看的主题，下载的主题都放在这里面，而主题子文件里面的目录结构以外层的基本一致
编辑配置文件

我们稍微按照自己的需要将config.toml编辑一下：

baseURL = "https://itachizhu.github.io/"
languageCode = "zh-cn"
title = "Golden Mountain"
[params]
  author = "itachizhu"
  subtitle = "这是一个码农Itachi的私人空间。"
安装Themes

使用git命令安装，实在因为网络等问题不行的画，可以自行创建文件夹然后把里面的东西复制进去。

$ cd themes
$ git clone https://github.com/halogenica/beautifulhugo.git
新增文章
$ hugo new posts/hugo.md

然后就是像我写这篇文章那样把这些内容写进去就行了。

预览

使用命令(在quickstart目录)：

$ hugo server --theme=beautifulhugo

然后访问http://localhost:1313即可看到效果。

打包

使用命令(打包前将public目录清空)：

$ hugo --theme=beautifulhugo
发布

将public目录里的所有文件上传到上面的github repository中去即可。当然可以使用git命令。
